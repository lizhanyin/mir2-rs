# 登录场景文档

## 概述

登录场景使用 `bevy_extended_ui` 框架实现 HTML/CSS 驱动的 UI，图片通过 `ImageCache` 缓存后在 CSS 中通过 `background-image` 使用。

### UI 图片 (Prguse.wzl)

|------|----------|------|

## ImageCache 图片缓存机制

### 1. 缓存 Key 格式

缓存 key 格式为 `{库名}_{索引}.png`，例如：
- `ChrSel_22.png` - ChrSel.wzl 中索引 22 的图片
- `Prguse_60.png` - Prguse.wzl 中索引 60 的图片

### 2. 加载图片到缓存

使用 `load_image_to_cache` 和 `load_images_to_cache` 函数：

```rust
use crate::resource::{load_image_to_cache, load_images_to_cache};
use bevy_extended_ui::ImageCache;

// 加载单张图片
let handle = load_image_to_cache(
    &config.resource_path,
    "data/ChrSel",      // 库名
    22,                 // 索引
    &mut *image_cache,
    &mut *images,
)?;

// 批量加载
let handles = load_images_to_cache(
    &config.resource_path,
    "data/Prguse",
    &[60, 61, 64, 53],  // 索引数组
    &mut *image_cache,
    &mut *images,
)?;
```

### 3. CSS 中使用缓存图片

图片加载到缓存后，可以在 CSS 中通过 `background-image` 引用：

```css
/* 使用缓存中的背景图片 */
body {
    background-image: url("ChrSel_22.png");
    background-position: center;
    background-size: cover;
    background-repeat: no-repeat;
}

/* 使用缓存中的表单背景 */
.login-box {
    background-image: url("Prguse_60.png");
    background-position: center;
    background-size: 100% 100%;
    background-repeat: no-repeat;
}
```

### 4. 工作原理

1. **加载阶段**：`setup_login()` 调用 `load_image_to_cache()` 从 WZL 文件加载图片
2. **缓存阶段**：图片被转换为 Bevy `Handle<Image>` 并存入 `bevy_extended_ui::ImageCache`
3. **渲染阶段**：CSS `background-image: url("ChrSel_22.png")` 被 `bevy_extended_ui` 解析
4. **查找阶段**：`get_or_load_image()` 在 `ImageCache.map` 中查找 key，找到则直接使用缓存的 handle

### 5. 关键代码

**bevy_image.rs** - 生成缓存 key：
```rust
pub fn make_cache_key(library_name: &str, index: usize) -> String {
    let name = Path::new(library_name)
        .file_name()
        .and_then(|n| n.to_str())
        .unwrap_or(library_name);
    format!("{}_{}.png", name, index)
}
```

**image_service.rs** (bevy_extended_ui) - 查找缓存：
```rust
pub fn get_or_load_image(
    path: &str,
    image_cache: &mut ImageCache,
    // ...
) -> Handle<Image> {
    if let Some(handle) = image_cache.map.get(path) {
        return handle.clone();  // 使用缓存
    }
    // ... 尝试从文件加载
}
```
