# 登录场景文档

## 概述

登录场景使用 `bevy_extended_ui` 框架实现 HTML/CSS 驱动的 UI，同时使用 Bevy Sprite 渲染背景图片。

## bevy_extended_ui 图片处理原理

### 1. 图片加载机制

**核心流程：**
```
HTML/CSS → 解析 → ImageCache → AssetServer → Handle<Image> → Bevy 渲染
```

**关键组件：**
- **ImageCache**：全局图片缓存资源，存储 `路径 → Handle<Image>` 映射
- **image_service.rs**：提供 `get_or_load_image()` 函数，负责加载和缓存图片

### 2. HTML 中使用图片的两种方式

#### 方式一：`<img>` 标签
```html
<img src="images/myimage.png" alt="描述">
```

#### 方式二：CSS background-image（推荐）
```css
.my-element {
    background-image: url("images/bg/test-bg.png");
    background-position: center;
    background-size: cover;
    background-attachment: scroll;  /* scroll | fixed | local */
}
```

### 3. 图片路径规则

图片路径相对于 **`assets/ui/`** 目录（由 `ExtendedUiConfiguration.assets_path` 配置）：

```rust
// main.rs 中配置
.insert_resource(ExtendedUiConfiguration {
    assets_path: ui_assets_path,  // 指向 assets/ui/
    ...
})
```

**示例：**
```css
/* CSS 中 */
background-image: url("images/bg/test-bg.png");
/* 实际路径: assets/ui/images/bg/test-bg.png */

/* 或使用绝对路径（从 assets 根目录） */
background-image: url("/ui/images/bg/test-bg.png");
```

### 4. 动态设置图片（代码方式）

如果需要在运行时动态设置图片（如从 WZL 库加载的图片），需要将图片添加到 `ImageCache`：

```rust
use bevy_extended_ui::services::image_service::get_or_load_image;
use bevy_extended_ui::ImageCache;

fn set_dynamic_image(
    mut image_cache: ResMut<ImageCache>,
    mut images: ResMut<Assets<Image>>,
    asset_server: Res<AssetServer>,
) {
    // 方式1: 通过路径加载（需要先保存到文件）
    let handle = get_or_load_image(
        "ui/form_bg.png",  // 相对于 assets 目录
        &mut image_cache,
        &mut images,
        &asset_server,
    );

    // 方式2: 直接添加已加载的图片到缓存
    // image_cache.map.insert("custom_key".to_string(), handle);
}
```

### 5. 图片使用方式对比

| 方式 | 适用场景 | 说明 |
|------|---------|------|
| `<img src="">` | 静态图片 | 简单直接 |
| `background-image: url()` | 背景/装饰 | 支持更多 CSS 属性 |
| `ImageCache` + 代码 | 动态图片 | 运行时生成/加载 |

### 6. 当前登录页面的实现方案

目前登录页面的背景图是通过 **Bevy Sprite** 单独渲染的，而非通过 HTML/CSS：

```rust
// src/scene/login.rs
// 生成背景精灵
commands.spawn((
    Sprite::from_image(image_handle),
    Transform::from_xyz(0.0, 0.0, -1.0)
        .with_scale(Vec3::new(scale, scale, 1.0)),
    LoginBackground {
        original_width: img_width,
        original_height: img_height,
    },
));
```

**如果要在 HTML 中使用 Prguse.wzl 的图片，可选方案：**

1. **导出为文件**：将 WZL 中的图片导出为 PNG，放到 `assets/ui/images/` 目录
2. **动态注入**：在代码中加载 WZL 图片，保存到临时文件或直接注入 `ImageCache`
